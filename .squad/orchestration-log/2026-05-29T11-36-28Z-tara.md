## Key Decisions

1. **Local dev validation:** Current platform only (< 1 min). AOT deterministic within platform.
2. **CI smoke tests:** linux-x64, osx-x64, osx-arm64 (native runners only).
3. **Build-only targets:** linux-arm64, win-x64, win-arm64 (cross-compile failures caught at build time).
4. **Blocker fallback protocol:** Explicit documentation in decisions.md with compiler evidence required before self-contained fallback.
5. **Expected CI time:** 3–5 min per PR (macOS steps slower but parallel).

## Risk Mitigation

- Cross-compile failures caught 99% at build time (trimmer, AOT compiler)
- Smoke tests exercise initialization (80% of AOT surprises occur here)
- Windows arm64 deferred (rare target, no free-tier native runner)
- Cost acceptable for MVP (macOS runners ~10× Linux cost; covers 10% user base)
